CSS Image Hover Effect

Through the use of the selector ":hover" one can apply a hover effect on an image element when a user hovers their mouse cursor over it. Along with the :hover selector, within the selection one needs to apply a "transition:" declaration to designate the property being affected, the length of time for the effect, the speed form of the effect, and if desired, the amount of time an effect is delayed before activating when the cursor begins to hover over the element.

As noted in the image below, for Figure 1, there has been a hover effect added for the image to lose opacity when a user hovers over it using the "opacity:()" property. Note the length of the effect, the speed of how the effect takes place, and the dramatic delay that was implemented.

When it comes to images, there are various effects one can apply to the image apart from their opacity. For example, through the use of the "transform" function, you can assign the "translate()" property so that one can designate the image to move a certain distance on the x-axis and/or y-axis when a cursor is over the image as noted below in Figure 2.

Through the use of the "scale()" property under the transform function, one can code an effect where the image scales in either the x-direction or the y-direction. Essentially, the scale property can "enlarge" or "shrink" an image. It can even flip a picture if the values are made negative. Below is Figure 4, showcasing how the opposite happens, the image enlarges quickly instead as instructed.

Apart from affecting the location and sizes of an image upon hovering, you can also add filter effects through the use of the "filter:()" to an image. For the filter property, there are different effects you can assign. For example, you can apply a grayscale percentage through the use of the "grayscale()" property, removing a certain amount of coloration from an image upon a hover.

Similarly, you can also apply a sepia filter by a percentage by using the "sepia()" property, changing the image's colors into a retro fashion yellow-brown and black coloration.

Another effect is to increase the saturation by a percentage using the "saturate()" property, strengthening the colors of an image. This can be used for emphasis on an image.

For those who seek a drastic effect, one can apply the "invert()" property to completely inverse the colors of an image to their opposites. Or, perhaps for the creative, you can apply this filter to "correct" an image back to normal colors upon the hover.

Naturally, apart from setting only one effect to take place, you can set multiple. To do this, make you sure are placing a comma between the sections of variables for each effect you are attempting to apply if you only wish to have specific effects to take place, otherwise, you can simply use the transition property without designating a particular effect. Using image hover effects can be quite fun, but one must use them sparingly as too many effects on an image can reduce the performance of a website on a user's computer. That being said, there is no shortage of creativity that one can show with their CSS code on an image hover.
